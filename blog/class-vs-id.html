A Differentiation Between Class and ID in CSS/HTML

10/09/2021

The Class and ID tags are attribute modifiers to specify the CSS styling of select element(s) in HTML. They are very frequently used and can technically be used interchangeably. However, there is a distinct best-practice in which to use these correctly.

Class

The class attribute can be used to specify the styling elements for one or more HTML elements. Rather than applying styling elements to each item, I can create a Class in CSS and apply that class to each element in HTML.

Syntax:

The class must be set in the opening – or first – tag in HTML and defined with a preceding “.” in CSS.


Example:

Say I wanted to add a box shadow to each div element I had, rather than specifying the styling element in each HTML <div> tag, I can specify a Class labelled ‘shadow’ – or whatever your heart desires, within reason – in CSS and apply it to each div tag I want.

I am using this method on this page to change the font-weight of the navbar items to 600 and add a hover animation to highlight the text. You can find this by opening the DOM and inspecting those items.

ID

ID, or name, works very much the same as class, however, it is best practice to use this to modify unique class elements, i.e., a single element.

Syntax:

The name is set in the opening tag in HTML of the element you wish to style and defined with a preceding “#” in CSS.
